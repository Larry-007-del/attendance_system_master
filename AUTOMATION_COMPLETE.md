## 🔧 How It Works

### Python Script Flow
```
START
  ↓
Scan directory for .md files
  ↓
For each file:
  - Count lines
  - Estimate pages (50 lines = 1 page)
  - Extract topics (### headings)
  - Build statistics
  ↓
Generate Markdown:
  - Overview sections
  - Statistics tables
  - Navigation guides
  - File listings
  ↓
Write DOCUMENTATION_INDEX.md
  ↓
Print statistics summary
  ↓
END
```

### GitHub Actions Flow
```
Git Event (push/PR)
  ↓
Check if .md files changed
  ↓
If yes:
  ↓
  Setup Python 3.12
  ↓
  Run update_docs_index.py
  ↓
  Check for changes
  ↓
  If changed:
    - Configure git
    - Add index file
    - Create commit
    - Push to repository
  ↓
Done!
```

### Issue: GitHub Actions not triggered
